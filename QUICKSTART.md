# 🚀 DHub Rejoin - Quick Start

**1️⃣ Open Termux in Redfinger**

**2️⃣ Copy-Paste this ONE command:**

```bash
curl -sSL https://raw.githubusercontent.com/turudek9991-sketch/Dhub-Rejoin/main/SETUP.sh | bash
```

**3️⃣ After setup, run:**

```bash
dhub
```
