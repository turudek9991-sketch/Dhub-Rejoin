🚀 DHub Rejoin - One-Command Installation

Jika repository sudah ada di GitHub (https://github.com/turudek9991-sketch/Dhub-Rejoin):

Copy-paste command ini di Termux:

═══════════════════════════════════════════════════════════════
curl -sSL https://raw.githubusercontent.com/turudek9991-sketch/Dhub-Rejoin/main/SETUP.sh | bash
═══════════════════════════════════════════════════════════════

Atau jika prefer manual download:

═══════════════════════════════════════════════════════════════
cd ~ && git clone https://github.com/turudek9991-sketch/Dhub-Rejoin.git && cd Dhub-Rejoin && lua54 dhub.lua
═══════════════════════════════════════════════════════════════

Atau paling simpel (tanpa git):

═══════════════════════════════════════════════════════════════
mkdir -p ~/Dhub-Rejoin && cd ~/Dhub-Rejoin && \
curl -sSL https://raw.githubusercontent.com/turudek9991-sketch/Dhub-Rejoin/main/dhub.lua -o dhub.lua && \
curl -sSL https://raw.githubusercontent.com/turudek9991-sketch/Dhub-Rejoin/main/README.md -o README.md && \
chmod +x dhub.lua && \
echo "alias dhub='cd ~/Dhub-Rejoin && lua54 dhub.lua'" >> ~/.bashrc && \
source ~/.bashrc && \
dhub
═══════════════════════════════════════════════════════════════

Setelah setup, cukup jalankan:

dhub

Done! Semua file sudah di-download dari GitHub, tidak perlu upload manual.
